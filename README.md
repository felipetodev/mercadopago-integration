## Installation

1. You'll first need to clone the repo, then install the dependencies:

```sh
npm install
```

2. Next, get token and public_key from `https://mercadopago.com/developers`:

```sh
TEST_ACCESS_TOKEN=
NEXT_PUBLIC_MERCADOPAGO_KEY=
```

3. Run the app:

```sh
npm run dev
```
